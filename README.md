# 💧 Bash Water Reminder

A simple hydration reminder for Linux using Bash.

## Features
- Desktop notifications
- Notification sound
- Custom interval
- Lightweight script

## Requirements

libnotify

Arch install:

sudo pacman -S libnotify

## Usage

chmod +x water_reminder.sh

Run with default 30 minute interval:

./water_reminder.sh

Run with custom interval:

./water_reminder.sh 20
